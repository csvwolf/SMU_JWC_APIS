# 上海海事大学教务处API

## 介绍

目前只使用Selenium，需要首先运行相关驱动（包括Selenium和PhantomJS，可以考虑自行集成）

## 用途
可以用于开发各种平台客户端版本，建议获取后将一些不常更新的数据存储在数据库。
